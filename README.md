## Design Highlights

| Feature | Details |
|---------|---------|
| **Typography** | Playfair Display + Geist Sans for an editorial, premium feel |
| **Color Palette** | Warm stone tones, cream backgrounds, forest green accents |
| **Animations** | Smooth page transitions, hover effects, parallax hero |
| **Responsive** | Mobile-first with animated slide-in navigation |
| **SEO** | Open Graph, Twitter cards, JSON-LD structured data |
